# Foro Hub

API REST de foro desarrollada con Spring Boot 3 como parte del challenge **Foro Hub** de Alura.

## Tecnologías

- Java 17+
- Spring Boot 3
- Spring Data JPA
- Spring Security + JWT
- Base de datos: (por definir)
- Maven

## Objetivo

Permitir la gestión de tópicos de un foro:
- Listar tópicos
- Crear, actualizar y eliminar tópicos (con autenticación JWT)
